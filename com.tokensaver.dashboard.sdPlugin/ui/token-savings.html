About the numbers

RTK is measured. The plugin reads rtk gain --all --format json, so the RTK, Today, Week, and Month readouts are real before/after token counts from RTK's own ledger — not estimates.

Graphify is auto-tracked but estimated. Install the track-graphify hook (see the README) and it counts each real graphify query into ~/.tokensaver/graphify.json and stamps your project's cost.json path — so both fields above are filled in automatically. The Graphify readout is net = estimated savings − real cost: queries × tokens/query (default 121,300, from the ~123k→1.7k benchmark) minus what cost.json says you spent. It's negative until queries pay back the build.

CodeGraph is auto-tracked but estimated. Install the track-codegraph hook (see the README) and it counts each real CodeGraph lookup — both the codegraph_explore/etc. MCP tools and the codegraph query CLI — into ~/.tokensaver/codegraph.json. CodeGraph builds its index 100% locally (no API cost) and records no savings of its own, so the readout is a pure realized estimate that climbs with use: queries × tokens/query (default 100,000). It's always marked ≈.

Total and Money blend measured RTK with the Graphify and CodeGraph estimates, so they're marked ≈. Money = saved tokens × $ per 1M tokens (default $3).
